# AIS_4G_EXTENSION_BOARD
library for 4G extension board v1.0 
Support sensor simply to use
- SHT20-x model
- Lightsensor BH1750
- Soil moisture (Capacitive Soil moisture v1.2 Analog sensor)
- I/O Expander PCA9557 (builtin)
- RTC PCF8563 (builtin)
